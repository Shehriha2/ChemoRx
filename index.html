Welcome To ChemoRx

Chemotherapy prescribing software

chemoRx is a Chemotherapy prescribing software. It is intended to provide a helping tool to prescribe a safe chemotherapy. It provides all healthcare professional with the help they need during prescribing chemotherapy. It provides all the calculations, transfere orders electronically to pharmacy, provides the support required to pharmacist including performing all the calculations, checking compabtibilities etc.
